Notes
- Les textures de pierre et de parchemin sont des bases visuelles générées par IA. Elles sont conçues pour servir de référence forte.
- Les éléments 9-slice sont dérivés automatiquement depuis les cadres générés.
- Les icônes sont sans texte et prêtes à être utilisées comme base de tabs.
- Quelques éléments peuvent nécessiter une légère retouche de production (nettoyage, vectorisation partielle, uniformisation de la lumière) avant intégration finale.
- Le pack vise surtout à servir de référence utilisable par les autres IA / graphistes du projet.
